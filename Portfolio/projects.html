Flappy Bird Game

This project presents a console-based implementation of Flappy Bird using the C programming language and Turbo C++’s graphics library. It simulates fundamental game mechanics such as gravity, obstacle movement, and collision detection through pixel-level rendering and timing controls. Designed for MS-DOS environments, the game showcases low-level graphical manipulation, event-driven programming, and creative logic structuring within a constrained compiler framework. .
